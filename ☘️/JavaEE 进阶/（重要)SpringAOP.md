# （重要)SpringAOP
> 相关笔记：[[JavaEE进阶|JavaEE进阶 知识总结]]


引入依赖

```java
<!--SpringBoot 3 版本-->
<dependency>
    <groupId>org.springframework.boot</groupId>
    <artifactId>spring-boot-starter-aop</artifactId>
    <scope>compile</scope>
</dependency>

<!--SpringBoot 4 版本-->
<dependency>
    <groupId>org.springframework.boot</groupId>
    <artifactId>spring-boot-starter-aop</artifactId>
    <version>4.0.0-M2</version>
    <scope>compile</scope>
</dependency>
```
