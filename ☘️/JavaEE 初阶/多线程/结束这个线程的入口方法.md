# 结束这个线程的入口方法
> 相关笔记：[[多线程|多线程]]


如何理解 “<u>这个线程的入口方法</u>” ？？

看Java中interrup方法源码

```java

  public void interrupt() {
        if (this != Thread.currentThread()) {
            checkAccess();

            // thread may be blocked in an I/O operation
            synchronized (blockerLock) {
                Interruptible b = blocker;
                if (b != null) {
                    interrupted = true;
                    interrupt0();  // inform VM of interrupt
                    b.interrupt(this);
                    return;
                }
            }
        }
        interrupted = true;
        // inform VM of interrupt
        interrupt0();
    }
```

这个方法就是使得线程处于是否为终止的状态，再通过isInterrupted（）方法返回的布尔变量对线程的入口方法进行某些判定，使得线程的入口方法执行完毕，这样就起到了终止/中断的效果
